JAWS Settings for LUWRAIN Application

Application settings that automatically enable JAWS sleep mode in a window of LUWRAIN self-voicing application.

Peculiarities

  • In the LUWRAIN application window, sleep mode of JAWS is enabled with unloading the synthesizer.
  • Braille sleep mode of JAWS does not enable, so the braille display keyboard can be used to enter commands and data in the LUWRAIN application window.
  • The settings are independent of the JAWS language and can therefore be used with any JAWS localization.
  • The settings support the LUWRAIN application running with both the embedded and locally installed JRE. However, to support a locally installed JRE, it may be necessary to rename the application settings file first. For more information, see "LUWRAIN run Options Support".

Installation

The application settings can be added to JAWS manually or automatically.

To add the application settings manually follow these steps:

  1. Open the JAWS user settings directory. To do this:
    1. Open the Windows Start menu.
    2. Move to the "All Programs" submenu and open it.
    3. Move to the group of used version JAWS and open it.
    4. Move to the "Explore JAWS" group and open it.
    5. Move to "Explore My Settings" item and activate it.
    Windows Explorer opens with a list of files and folders (for the current user) containing user settings of JAWS.
  2. Copy all luwrain*.JCF files in this package to the JAWS user settings directory.

The luwrain.JCF file is for the LUWRAIN application running with the embedded JRE, and the remaining luwrain*.JCF files are for the LUWRAIN application running with the locally installed JRE. However, to support a locally installed JRE, it may be necessary to rename the application settings file first. For more information, see "LUWRAIN run Options Support".

To add the application settings automatically run the install.vbs script (or one of the script localizers, for example, Russian install_ru.vbs, for installation with messages in the appropriate language). The script will add the application settings to all JAWS versions detected on the system and for all JAWS localization languages.

The automatic installation script is compatible with both windows based and console based script host. For greater automate the installation process without blocking dialog boxes, the main script (or any script localizer) can be run using the CScript script host, for example, with commands: 

CScript install.vbs //Nologo
CScript install_ru.vbs //Nologo //U

LUWRAIN run Options Support

The LUWRAIN application can be run in two ways:

  1. Running LUWRAIN with the embedded JRE: The recommended default option, which runs luwrain.exe via a shortcut or by directly running the executable file. The settings from the luwrain.JCF file are intended for this run option.
  2. Running LUWRAIN with the locally installed JRE: An option for special needs and advanced users, which an executable JAR file is run using a separately Java Runtime Environment, for example, with a command: 
    javaw -jar jar\luwrain-base.jar
    The settings from the luwrain-base.JCF file are intended for this run option.
    The name of the executable JAR file with the main class may change in different builds of LUWRAIN, so it is necessary to rename the settings file accordingly before installing it. If necessary, you can create several additional settings files (the automatic installation script copies them with wildcards as luwrain*.JCF).

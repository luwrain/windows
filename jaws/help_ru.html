Настройки JAWS для приложения LUWRAIN

Настройки для приложения, которые автоматически включают спящий режим JAWS в окне самоозвучивающегося приложения LUWRAIN.

Особенности

  • В окне приложения LUWRAIN включается спящий режим JAWS с выгрузкой синтезатора.
  • Спящий режим JAWS для брайля не включается, поэтому клавиатура брайлевского дисплея может использоваться для ввода команд и данных в окне приложения LUWRAIN.
  • Настройки не зависят от языка JAWS, поэтому могут быть использованы с любой локализацией JAWS.
  • Настройки поддерживают приложение LUWRAIN, запускаемое как со встроенной, так и с локально установленной JRE. Однако для поддержки локально установленной JRE может потребоваться сперва переименовать файл настроек приложения. Подробнее см. «Поддержка вариантов запуска LUWRAIN».

Установка

Настройки для приложения могут быть добавлены в JAWS вручную или автоматически.

Чтобы добавить эти настройки для приложения вручную, выполните следующие действия:

  1. Откройте директорию пользовательских настроек JAWS. Для этого:
    1. Откройте меню «Пуск» Windows.
    2. Перейдите к подменю «Все программы» и откройте его.
    3. Перейдите к группе используемой версии JAWS и откройте её.
    4. Перейдите к группе «Файлы JAWS» и откройте её.
    5. Перейдите к пункту «Мои настройки» и активируйте его.
    Откроется Проводник Windows со списком файлов и папок (для текущего пользователя), содержащим пользовательские настройки JAWS.
  2. Скопируйте все файлы luwrain*.JCF из этого пакета в директорию пользовательских настроек JAWS.

Файл luwrain.JCF предназначен для приложения LUWRAIN, запускаемого со встроенной JRE, а остальные файлы luwrain*.JCF для приложения LUWRAIN, запускаемого с локально установленной JRE. Однако для поддержки локально установленной JRE может потребоваться сперва переименовать файл настроек приложения. Подробнее см. «Поддержка вариантов запуска LUWRAIN».

Чтобы добавить эти настройки для приложения автоматически, запустите скрипт install.vbs (или один из локализаторов скрипта, например, русский install_ru.vbs, для установки с сообщениями на соответствующем языке). Скрипт добавит эти настройки для приложения во все обнаруженные в системе версии JAWS и для всех языков локализации JAWS.

Скрипт автоматической установки совместим как с оконным, так и с консольным сервером сценариев. Для большей автоматизации процесса установки без блокирующих диалоговых окон основной скрипт (или любой локализатор скрипта) можно запускать с использованием сервера сценариев CScript, например, командами: 

CScript install.vbs //Nologo
CScript install_ru.vbs //Nologo //U

Поддержка вариантов запуска LUWRAIN

Приложение LUWRAIN может быть запущена двумя способами:

  1. Запуск LUWRAIN со встроенной JRE: Рекомендуемый вариант по умолчанию, при котором запускается luwrain.exe с помощью ярлыка или путём прямого запуска исполняемого файла. Для этого варианта запуска предназначены настройки из файла luwrain.JCF.
  2. Запуск LUWRAIN с локально установленной JRE: Вариант для особых надобностей и опытных пользователей, при котором запускается исполняемый JAR-файл с использованием отдельной среды выполнения Java, например, командой: 
    javaw -jar jar\luwrain-base.jar
    Для этого варианта запуска предназначены настройки из файла luwrain-base.JCF.
    Название исполняемого JAR-файла с основным классом может меняться в разных сборках LUWRAIN, поэтому требуется соответственно переименовывать и файл настроек перед его установкой. При необходимости можно создать несколько дополнительных файлов настроек (скрипт автоматической установки копирует их с подстановочными знаками как luwrain*.JCF).
